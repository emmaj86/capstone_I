What is web development?

Web development refers in general to the tasks associated with developing websites for hosting via intranet or internet. The web development process includes web design, web content development, client-side/server-side scripting and network security configuration, among other tasks (Techopedia). Web developers work with many languages. Some only work with front end which is the client side, like a website. Back end developers work with the server side which is what users cant see like databases of users information for example. You also get full stack developers who work with both back end and front end applications.
